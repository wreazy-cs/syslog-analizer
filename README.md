# syslog-analizer
Linux syslog kayıtlarını dosyadan okuyup çift yönlü bağlı liste (doubly linked list) ile saklayan ve üzerinde ileri/geri gezinme, anahtar kelime filtreleme, süreç bazlı analiz ve istatistik çıkarma işlemleri gerçekleştiren C uygulaması. Kırklareli Üniversitesi Veri Yapıları dersi ödevi.
